h5version {rhdf5}R Documentation

rhdf5 and libhdf5 version numbers

Description

Returns the version number of the Bioconductor package rhdf5 and the C-library libhdf5.

Usage

h5version()

Value

A list of major, minor and release number.

Author(s)

Bernd Fischer
